You need to navigate through the different folders to inspect the files and decode what the next web address will be. Below is a list of commands you will need to use. Research these Unix terminal commands to figure out what they do and how to use them.

  • pwd - print working directory (displays full path of the current folder)
  • ls - list (displays contents of the current folder)
  • cd - change directory (changes your current folder)
  • cat - catenate (displays contents of file)
