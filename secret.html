The flag is encoded inside a large data file so that no one else can uncover the message. The data file is located here.

The flag has been inserted one character at a time in order throughout the text file. Each character of the flag is always preceded by the preamble ?msg?.

Look at the example data provided below. You can see there are 2 ?msg? preambles in the data. The first is followed by an h and the second is followed by an i. In this case, the flag would be hi.
